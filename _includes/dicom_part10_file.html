DICOM Part 10 File
Preamble (128 Bytes) — Offset 0x00
Spec: "This Part of the DICOM Standard does not require any structure for this fixed size Preamble."

Intended: dual-format headers (TIFF, etc.)
Actual: free parking — unchecked by design.
DICM Magic (4 Bytes) — Offset 0x80
Value: 0x44 0x49 0x43 0x4D ("DICM")
The only validated field in the header. Everything before it is unchecked.
File Meta Information Group — Offset 0x84+
Group (0002,xxxx) — explicit VR Little Endian, always.
Transfer Syntax UID (0002,0010) tells the parser how to read the rest.
Dataset (Variable Length)
Flat stream of Tag-Length-Value data elements, all the way to EOF.
