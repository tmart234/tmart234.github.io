DICOM Data Element (Explicit VR LE)
Tag (4 Bytes)
Group Number (2) | Element Number (2)
Odd group numbers = private / vendor-defined.
VR (2 Bytes) + Length (2 or 4 Bytes)
VR is two ASCII chars: OB, OW, UI, SQ, UN, etc.
OB/OW/OF/SQ/UT/UN use 4-byte length. Values up to 2³² − 1.
Value Field (Length Bytes)
Where Payloads Hide
Pixel Data (7FE0,0010) — VR=OB
Encapsulated frames — arbitrary bytes between expected image data.
Private Tags — Odd Groups
Spec requires viewers to tolerate unknown private elements. In practice: ignored.
Overlay Data (60xx,3000) VR=OW — bitmap planes, rarely content-validated.
Waveform Sequence (5400,0100) VR=SQ — nested TLV streams inside.
Parsers check TLV structure, not value semantics. Every container above is a payload opportunity.
